# Triple Review

Pre-PR Triple Review System — CodeRabbit, Claude CLI, Codex CLI の3ツール並列コードレビュー。

## Quick Setup

```bash
# 1. Clone
git clone <this-repo> ~/workspaces/triple-review

# 2. Symlink
ln -sf ~/workspaces/triple-review ~/.config/triple-review

# 3. Install
bash ~/.config/triple-review/install.sh
```
